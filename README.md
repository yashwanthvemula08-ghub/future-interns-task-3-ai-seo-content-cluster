# AI SEO Blog & Content Cluster Generator for Business Websites

## Overview
This repository demonstrates prompt engineering techniques for generating SEO-optimized website content and content clusters using AI. The prompts are designed to create professional, search-engine-friendly content for local businesses.

## Objective
- Generate SEO-friendly website content.
- Create engaging blog articles.
- Build an effective content cluster strategy.
- Demonstrate AI-assisted prompt engineering for digital marketing.

## Repository Structure
```
README.md
prompts.md
outputs.md
screenshots/
```

## Tools Used
- ChatGPT
- GitHub
- Markdown

## Files
- **prompts.md** – Contains all prompts used to generate SEO content.
- **outputs.md** – Contains the AI-generated website content and blog articles.
- **screenshots/** – Includes screenshots of the prompt engineering process.
